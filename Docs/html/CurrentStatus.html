CurrentStatus

For Freevo 2.0 we changed many parts of the code, some parts moved in extra python modules and some parts got a complete rewrite. Because of that, not everything is working right now, e.g. the gui part got a restructuring with a big rewrite of important parts. For that reason some plugins still don't work, they are auto deactivated and can only be activated when the code is ported to the new interface.

Changes

  • Changed the main loop to use PyNotifier. No threads are allowed anymore and every part must take care that the notifier loop is kept alive. Not all parts respect that right now.

  • Removed Twisted. It is a huge monster and hard to understand. For inter-process communication a python implementation (PyMbus) of the Mbus transport protocol is used. As a side effect, modules using twisted need to be rewritten.

  • Complete rewrite of the recordserver.

    • Switched to Mbus

    • Support more than one tv card

    • Conflict detection

  • Rewrote the channels part of the tv module. It now uses a database as backend for much faster access. This whole epg part is in an extra package PyEPG

  • Major cleanup of the gui, skin and osd part. Everything gui related is now in the gui subdirectory

    • Moved to Mevas as gui backend, using imlib2 as underlaying render engine

    • Support different display engines. Besides the old pygame output you can now also use native imlib2 for X, DirectFB (PleaseUpdate: status of DirectFB?) or bmovl and bmovl2 on mplayer (currently broken as primary backend).

    • Draw with the normal Freevo gui code on mplayer using bmovl

Status

Most parts of Freevo already work again. There are at least three people using cvs Freevo for daily use. But it may crash. Here a list of parts that won't work:

  • The tv guide is slow and has a bad support for the recordserver. In most cases you need to edit the recordings in the fxd file after scheduling. The tv part is under heavy development

  • Some plugins are deactivated

  • The webserver doesn't work because we removed Twisted

Everything else should work.
