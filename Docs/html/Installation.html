Installation

First of all, you should subscribe to the Freevo developer mailinglist to get usefull informations of the current status of Freevo.

Using the source tree

To test Freevo, use the directory you checked out in. Before you can start Freevo, you need to type 'makde'. This will compile some modules in the lib tree and download external dependencies like mbus and the notifier. As a new requirement (besides the Freevo 1.5 modules), you need Imlib2 and pysqlite.

Installing Freevo

This is tricky right now. There are no ebuilds, RPMs, Debian packages or similar. Go into the lib directory and call 'python setup.py install' as root on all packages. After that, you can call 'python setup.py install' in the root directory. 

cd ~/src/freevo/lib
for d in pyimlib2 mevas pyNotifier-0.3.0 pyMbus-0.8.2 pyepeg; do
   (cd $d && python setup.py install)
cd ..
python setup.py install
